GIT Console comands


Git exploración:
ls: ver carpeta.
ls -a: ver carpeta oculta.
dc: retroceder carpeta.
clear: limpia la consola

Git usuario:
git config --global user.name (nombre x): establece el nombre del usuario.
git config --global user.mail (mail x): establece el mail del usuario.

Manejo de Git:

IMPORTANTE
git init: inicializar repositorio.
git status: comprobar cambios en el repositorio.
git add (archivo x): actualiza el archivo para el repositorio.
git commit -m "(anuncio)": sube una copia del repositorio.

git log --oneline: mejora la comprensión del código.
git branch (nombre de la rama): crea una rama (por defecto existe "master").
git branch -d (nombre de la rama): elimina la rama.
git checkout (nombre de la rama): para cambiar la rama en la que se trabaja actualmente.
